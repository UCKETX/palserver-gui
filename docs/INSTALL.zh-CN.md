# palserver GUI 安装与连接指南(玩家版)

[繁體中文](INSTALL.zh-TW.md) | **简体中文**

这份是给「想开一台 Palworld 服务器」的玩家看的。不需要懂程序,照着做就好。

---

## 一、下载

到 [Releases 页面](https://github.com/UCKETX/palserver-gui/releases) 下载对应你系统的文件:

| 你的系统 | 下载 | 内容 |
| --- | --- | --- |
| Windows | `palserver-agent-windows.zip` | 免安装可执行文件 + 网页界面(最常用) |
| Linux | `palserver-agent-linux.zip` | 同上 |
| macOS | `palserver-agent-macos.zip` | 同上(Mac 无法实际跑 Palworld 服务器,仅供管理远程主机) |

解压缩后会看到:

```
palserver-agent(.exe)   ← 主程序,双击运行
web/                    ← 网页界面(别删,程序会自动用它)
```

> **不用另外安装 Node.js 或任何东西**,可执行文件已经把需要的都包进去了。

---

## 二、启动

双击 `palserver-agent.exe`(Windows)或在终端机运行 `./palserver-agent`(Mac/Linux)。

- Windows 第一次运行,SmartScreen 可能跳「不明发行者」警告 → 按「其他信息」→「仍要运行」。
- 窗口会保持开着,关掉窗口 = 关掉服务器管理服务,所以先让它开着。

启动后窗口会显示类似这样的消息:

```
  ┌───────────────────────────────────────────────
  │  palserver GUI agent 已启动
  │
  │  本机管理(免密码,直接打开):
  │      http://localhost:8250
  │
  │  同一局域网 / VPN 的设备可连:
  │      http://100.86.188.111:8250   (Tailscale)
  │      http://192.168.0.19:8250
  │
  │  邀请朋友远程连接 —— 把这条链接传给他:
  │      http://100.86.188.111:8250/?setup=EEHN-CVMY
  │  或请对方在网页输入配对码:EEHN-CVMY
  └───────────────────────────────────────────────
```

---

## 三、开始管理(在自己这台电脑)

直接用浏览器打开 **http://localhost:8250** —— 会直接进入管理接口,**不需要输入任何密码**(本机连接自动信任)。

接着就能创建服务器、改设置、备份、看在线玩家等。

---

## 四、让朋友一起玩

Palworld 专用服务器要能被朋友连到,有两条路:

### 路线 A:VPN(推荐,最简单也最安全)

你和朋友装同一套免费 VPN、加入同一个网络,就像在同一个 WiFi 里,不用动路由器设置。

- **Radmin VPN** — 免注册,建个房间邀朋友加入,最适合游戏联机。
- **Tailscale** — 用 Google/GitHub 帐号登录,稳定,适合长期。

装好 VPN 后,把 agent 窗口上的 **VPN 地址**(通常是 `100.x.x.x`)给朋友。他要进「管理接口」的话,你再把 `?setup=配对码` 那条链接给他即可。

> **VPN 本身就是加密渠道**,所以就算走的是 `http`,在 VPN 内也是安全的。

### 路线 B:公开 IP 直连(高端)

需要处理路由器端口转发、防火墙、浮动 IP / CGNAT 等,新手较麻烦。我们有提供「IP 直连设置服务」协助设置,详见接口内说明。

## 五、关于管理接口的连接与安全

| 情境 | 会怎样 |
| --- | --- |
| 自己这台电脑开 `localhost` | 直接进,免密码 |
| 同局域网/VPN 的另一台设备 | 需要**配对码**(agent 窗口上有),或点 `?setup=` 链接一次即可记住 |
| 公开网站版(未来)连自己家的 agent | 受浏览器限制,建议走 VPN 或合一版本机管理 |

高端选项(环境变量):

- `PALSERVER_REQUIRE_TOKEN=1` — 连本机也强制要配对码(多人共用的电脑建议开)。
- `PALSERVER_TLS=1` — 用 HTTPS 监听(自签凭证会跳浏览器警告;想要公开直连的加密才需要)。
- `PALSERVER_AGENT_PORT=8250` — 改端口。
- `PALSERVER_WEB_ORIGINS=https://你的公开站` — 允许某个公开网站跨来源连这台 agent。

## 六、常见问题

**朋友连不进游戏服务器?**
先确认服务器正在运作中、防火墙有放行「游戏端口(UDP)」,以及朋友用的是正确的地址。

**打开网页看到的是旧画面?**
更新 agent 后按一次 `Ctrl+Shift+R`(强制刷新)。

**配对码在哪?**
在 agent 启动的窗口里。忘了可以在管理接口重新产生。

**要换一台电脑管理?**
把 agent 的 VPN 地址 + 配对码给那台设备即可,存档和设置都在跑 agent 的主机上。
